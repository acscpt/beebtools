# beebtools documentation

Detailed reference for the `beebtools` BBC Micro DFS disc image tool.
For a quick overview and installation instructions, see the
[project README](../README.md).

## Command reference

| Command | Description |
| --- | --- |
| [cat](commands/cat.md) | List disc catalogue with file types and metadata |
| [search](commands/search.md) | Search BASIC source for a text pattern or regex |
| [extract](commands/extract.md) | Extract a single file or bulk-extract all files |
| [create](commands/create.md) | Create a blank formatted disc image |
| [add](commands/add.md) | Add a file to an existing disc image |
| [delete](commands/delete.md) | Delete a file from a disc image |
| [build](commands/build.md) | Build a disc image from files with .inf sidecars |
